# Mushroom App

Production tracking for small mushroom farms — full bag lifecycle from liquid culture → spawn → substrate → harvest → disposal.

## Stack

- **Frontend:** Next.js (TypeScript)
- **Backend:** FastAPI (Python 3.12)
- **Database:** PostgreSQL (prod), SQLite (dev/test)
